<h2>Steps:</h2>

1) Clone repo
```
git clone https://github.com/Jusav503/PruebaTecnica.git
```

2) Install composer dependecies

```
composer install
```

3) Install NPM Dependencies
```
npm install
```

4) Create a copy of your .env file
```
cp .env.example .env
```
5) Generate an app encryption key
```
php artisan key:generate
```
